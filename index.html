FORMS CHEATSHEET TABLE

Tag

Atribute

Use
< label for > Label Is the text shown on the form.
< Form> METHOD= GET Is used to create a form where you get the data from the server.
METHOD= POST Is used to create a form where you send data to the server.
< input> Type: text Use it to add a text box to your form.
Type: range You can add a slider with this input type.
Type: password Is used to add a box for hidden text (passwords, security questions, etc.).
Type: number Is used for a box to accept only numbers.
Type: radio Radio is better used when the user is presented with multiple choices where he/she has to select only one.
Type: date Creates a box that selects a specific date.
Type: Checkbox Better to use it when the person has to select multiple choices.
Type: textarea Used to create a text box, for example, recommendations, allergies, etc.
Type: list Create a dropdown list with the predetermined values of the < datalist >.
type: submit Creates the button to submit your information.
< datalist > < option > Create a dropdown list with predetermined values.
< Select> < option value> Creates a dropdown list of option where the user has to select one.

Things to remember

  • Step: it is used to set the velocity of the type: range slider, number, etc...

  • Name: Description of the input.

  • Required: Make the user to select one option mandatory.

  • Min, Max: Min or max number.

  • Minlength, Maxlength: Set a minimum or max length for your box.

  • Pattern: [a-zA-Z0-9] With this Atribute you can set the characters allowed in the box.
